What is it?

Dash Reports is a veristile and light weight reporting solution. It allows users and administrators to create a large range of reports driven by SQL databases.

It also supports the creation of complex and powerful dashboards; caching the results locally to reduce load on the source data.

Various report output formats are supported and the results can be styled using templates.

The system allows for scheduled or user invoked reports, delivered via browser, file, FTP/SFTP or email.
Key Features
  • Secure
  • Flexible
  • Templatable output
  • Multiple Delivery Options
  • Powerful dashboards
  • Full auditing
  • Output and parameter bursting
  • Fully open-source (GPL v3)
Technology

Dash Reports is based on industry standard technologies and is simple to support and deploy. Some of the technologies are listed below.

  • Java
  • Spring
  • Hibernate
  • Quartz
  • Jasper Reports
  • jFree Charts
  • jQuery
Ease of Use

Dash Reports is designed to be simple. It is based around the principle that the SQL query is designed in another tool and then inserted into the report or dashboard definition.

It has easy to navigate menus and a simple and clean user-interface. Administration is entirely UI based (with the exception of the server parameters) and is fully secured.

Access to reports and dashboards is based on group membership. Users can also be created as readonly, with no permissions to change reports or dashboard items. Groups are managed by the system administrator through the administration UI.
