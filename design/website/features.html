Key Features

SQL Driven Reports and Dashboards

Dash Reports allows users and administrations to create powerful SQL driven reports and dashboards.

The dsahboards can be either: chart, grid, threshold or sampling. They are highly customisable.

Reports are very powerful and allow bursting of parameters and output destination. In addition various output formats and destinations are support.

Powerful Scheduling Options

Schedule your reports or dashboards at any possible interval, with granularity down to a second. The scheduler is powered by the industry strength Quartz Scheduler.

Multiple Output Formats and Options

Supported output formats are: HTML, Excel (Single and tabbed), CSV and PDF.

Strong Security Framework

Security is based around the Spring Security framework. Encryption is used to secure both user and datasource passwords.

Extensive Administration Options

You can administer the application via the powerful management console;available to administrator level users. From here you can add: users, datasources and groups - and also monitor the system via the powerful event framework.

Detailed Auditing

Every action in the system is audited and this can be viewed by a detailed console. The console can focus in on specific areas and select on a time-frame. Each action is also timed for performance, allowing you to ensure you have no long-running jobs or dashboards.

Sampling Data Dashboard Function

Sampling Dashboards allow you to define a piece of SQL that will be run at a specific interval. The result of this will be "sampled", recorded and tracked on a chart. This can be configured to sample every second, minute, hour etc and has a definable window. The system can also collect trend data and plot minimum, maximum and mean values for the sampled data.

No Browser Add-ons required

The user-interface is pure HTML. There is no need to install plugins such as Java or Flash on the client machines/

Open Source

Dash Reports is 100% open-source. You do not have to pay a penny to download, install and use it. The source code is available for review (on G6oogle Code). It is licensed via the GPL v3 public license.
